1. Выбери директорию на диске, в которой будет хранится проект. Например, папка hugo
2. Скопирую в нее файл init.bat
3. Запусти файл init.bat
    - в директории должна создаться папка nla_site
4. Переходишь в папку nla_site. Там файлы
    - dev_site.bat - старт проекта для локальной разработки. Стартует локальный сервер и сайт открывается локально
    - deploy.bat - выкатывание обновленной версии сайта на сервер. Чтобы каждый раз не вбивать username и password от гитхаба
    можешь в комндной строке выполнить команду:

    git config —global credential.helper wincred

    это кэширование введенного имени и пароля

5. папки для редактирования контента
    - nla_site/content. Здесь по типам контента. Например для добавления нового курса нужно создать новую папку в директории content/course
    - nla_site/themes/nla/layouts/index.html - это редактирование главной страницы


git fetch --all
git reset --hard origin/master
